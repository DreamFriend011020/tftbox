# TFT.GG Project

## Overview
TFT.GG is a web application designed to provide players with strategies and information for the game Teamfight Tactics (TFT). The website features tier lists, champion information, and item combinations to help players improve their gameplay.

## Project Structure
```
tft-workspace
├── src
│   ├── tft_main.html        # Main HTML structure for the website
│   ├── css
│   │   └── tft_main.css     # Styles for the HTML elements
│   ├── js
│   │   └── main.js          # JavaScript for interactivity
│   └── data
│       └── champions.json    # Data related to champions
├── .vscode
│   └── settings.json        # Workspace-specific settings for VS Code
├── .gitignore               # Files and directories to ignore by Git
├── package.json             # npm configuration file
└── README.md                # Documentation for the project
```

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install the necessary dependencies using npm:
   ```
   npm install
   ```
4. Open `src/tft_main.html` in your web browser to view the application.
